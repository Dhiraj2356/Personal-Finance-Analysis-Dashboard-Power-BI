# Personal-Finance-Analysis-Dashboard-Power-BI

## Project Overview

This project focuses on analyzing personal finance data to track Income, Expenses, and Savings on a monthly and yearly basis.
The goal was to create an interactive Power BI dashboard that provides clear financial insights for better decision-making.

 ## Objectives

Track Total Income, Expenses, and Savings
Analyze category-wise spending and saving
Monitor Savings Target (25% of Income)
Provide KPI-based financial overview


## Key KPIs Created (Using DAX)

Total Income
Total Expenses
Total Savings
Savings Percentage
Expense Percentage
Monthly Savings Target (25% of Total Income)

## Dataset

Source: Public dataset from GitHub 
Data includes income sources, expense categories, dates, and amounts

## DAX Functions Used

SUM()
CALCULATE()
DIVIDE()

These were used to calculate totals, percentages, and conditional metrics.

## Key Learnings

Writing efficient DAX measures
Validating dashboard numbers
Designing KPI-driven reports

Practical understanding of financial reporting
